## Bangbezh
Bangbezh is an athan app for Kurdistan.

### Main Features:
+ Prayer times for 13 Kurdistan cities
+ Can call for prayer in many reciters.
+ Can show the prayer time for the next 10, 15 or 30 days
+ Azkar
